Peer Instruction
for very large Groups

Pingo ist ein kostenloses Live-Feedback-System für die Lehre zur Aktivierung von Studierenden insbesondere in großen Lehrveranstaltungen.

Als Dozent anmelden mehr erfahren ›
{% include nav.html %}

Keine Installation, keine zusätzliche Hardware notwendig.

PINGO funktioniert über den Browser und wird von der Universität Paderborn kostenlos zur Verfügung gestellt. Studierende können einfach über den Browser in ihrem Smartphone, Tablet oder Notetbook abstimmen. Ebenso können Dozenten ihre Fragen über den Browser vorbereiten und Umfragen starten bzw. stoppen.

Legen Sie sofort los

Es ist keine Vorbereitung notwendig. Melden Sie sich an und starten sie direkt eine Quick-Session, um unmittelbares Feedback zu bekommen.
