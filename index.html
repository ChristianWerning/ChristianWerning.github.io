Peer Instruction
for very large Groups

PINGO is a free to use live-feedback-system for education, especially in large groups.

Sign up as lecturer learn more ›
{% include nav.html %}

Keine Installation, keine zusätzliche Hardware notwendig.

PINGO funktioniert über den Browser und wird von der Universität Paderborn kostenlos zur Verfügung gestellt. Studierende können einfach über den Browser in ihrem Smartphone, Tablet oder Notebook abstimmen. Ebenso können Dozenten ihre Fragen über den Browser vorbereiten und Umfragen starten bzw. stoppen.

Start immediately

Es ist keine Vorbereitung notwendig. Melden Sie sich an und starten sie direkt eine Quick-Session, um unmittelbares Feedback zu bekommen.
