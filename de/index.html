Peer Instruction
for very large Groups

PINGO ist ein kostenloses Live-Feedback-System für die Lehre zur Aktivierung von Studierenden insbesondere in großen Lehrveranstaltungen.

Als Dozent anmelden mehr erfahren ›
{% include nav.de.html %}

Keine Installation, keine zusätzliche Hardware notwendig.

PINGO funktioniert über den Browser und wird von der Universität Paderborn kostenlos zur Verfügung gestellt. Studierende können einfach über den Browser in ihrem Smartphone, Tablet oder Notebook abstimmen. Ebenso können Dozenten über den Browser Umfragen starten bzw. stoppen und ihre Fragen vorbereiten.

Legen Sie sofort los

Sie können PINGO unmittelbar nach Ihrer Registrierung vollumfänglich einsetzten. Es ist keine Vorbereitung notwendig, starten Sie Ihre erste Umfrage direkt als Quick-Session.
