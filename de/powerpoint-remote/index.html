PINGO Remote

Steuern Sie Ihre PINGO-Umfragen direkt von Ihren Folien aus

PINGO Remote ist eine kleine Anwendung, die in einem Fenster im Vordergrund läuft. Sie können es mit jeder Präsentationssoftware oder PDF-Folien nutzen. Mit PINGO Remote benötigen Sie keinen Browser mehr, um Umfragen zu starten und die Ergebnisse anzuzeigen.

PINGO Remote zurzeit nur für Windows erhältlich.

Kostenloser Download

Windows .exe Format, 65mb

Features

  • Kostenlos und Open Source
  • Starten und Stoppen von Umfragen
  • Erstellung von Quick-Sessions
  • Laden Sie vordefinierte Fragen in Ihre Sessions
  • Sehen Sie Ihre Ergebnisse direkt am Ende der Umfrage

System-Voraussetzungen

  • Windows 7/8, kein RT
  • Internetverbindung

Aktuelle Version: RC2 (veröffentlicht am 24. Mai 2014)
